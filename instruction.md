Implementation Plan: React-to-Android via Capacitor1. Project ConfigurationThe following files have been created/modified to support the Android build:•capacitor.config.ts: Configures Capacitor to use the dist folder (Vite's default output) as the source for the mobile app.•package.json: Added convenience scripts:◦npm run android:setup: Installs dependencies and initializes the native Android project.◦npm run android:run: Builds the React app, syncs assets to Android, and opens Android Studio.2. Environment Variables (Critical)Since this is a mobile app, .env files work differently than on a web server.•Current State: Your vite.config.ts bakes the GEMINI_API_KEY into the build during npm run build.•Action: Ensure your .env file contains your GEMINI_API_KEY before running the build command, otherwise, the AI features won't work in the emulator.3. Execution StepsStep A: Initialize the Native Layer Run this once to generate the android/ directory:npm run android:setupStep B: Build and Deploy Run this whenever you want to test your latest code on the emulator:npm run android:run4. Manual Steps in Android StudioOnce Android Studio opens the project:1.Wait for Gradle Sync: Look at the bottom bar; wait for "Gradle Sync Finished".2.Select Target: In the top toolbar, ensure an Emulator (e.g., Pixel 7) is selected.3.Run: Press the Green Play Button (Shift + F10).Technical "Prompt" for Implementation TrackingIf you need to explain this implementation to another developer or AI, use this:"I am converting a Vite/React web application into a native Android app using Capacitor. The project uses Tailwind CSS and Gemini AI. The implementation strategy involves: 1) Creating a capacitor.config.ts pointing to the dist web directory. 2) Adding @capacitor/core, @capacitor/cli, and @capacitor/android to the project. 3) Automating the workflow with npm scripts that run vite build followed by cap sync android. 4) Handling API keys via Vite's define property during the build phase so they are embedded in the mobile bundle. The final output is a native Android Studio project located in the /android directory."
